# 📊 Student Lifestyle, Study Habits, and Academic Performance

An interactive data visualization project exploring how lifestyle factors such as study time, sleep, motivation, and access to resources influence student exam performance.

🔗 **Live Website:**  
https://livia-cutra.github.io/sleep-student-performance/

## 🎯 Project Overview

This project investigates the relationship between student behaviors and academic outcomes using interactive visualizations. 

We designed a narrative-driven website that guides users from **overall distributions** to **group comparisons** and finally to **multi-variable relationships**, making patterns in the data easier to understand.

---

## 🧠 Key Questions

- How do study habits and sleep impact exam performance?  
- What role does motivation play in academic success?  
- How do resources and school type influence outcomes?  
- How do multiple factors interact together?  

---

## 📈 Visualizations

This project includes a combination of **Altair (Vega-Lite)** and **D3.js** visualizations:

### Altair (Vega-Lite)
- Histogram (exam score distribution)  
- Faceted chart (motivation comparison)  
- Line chart (resources vs. performance trends)  
- Box plot (study time vs. score distribution)  
- Scatterplot (study hours vs. exam scores)  

### D3.js
- Grouped bar chart (public vs. private comparison)  
- Heatmap (study + sleep interaction)  
- Treemap (score distribution by group)  

---

## 🎨 Design Approach

Our design focused on:

- **Clarity** → consistent labels, layouts, and figure structure  
- **Comparison** → side-by-side charts and shared scales  
- **Storytelling** → logical flow from overview → comparison → relationships  
- **Usability** → “How to Read” cards and simplified interactions  

We also incorporated feedback to improve:
- chart readability  
- layout structure  
- color consistency  
- and interaction design

## 💡 Key Takeaways

- Academic performance is influenced by multiple interacting factors
